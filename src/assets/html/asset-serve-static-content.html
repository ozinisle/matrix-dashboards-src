External HTML Content Loading

Following method lets you include external html files into your code directly

1. create your component html with the following code <div [innerHtml]="myTemplate"> </div>

2. create your component with the following code. This should do.

    export class ServeStaticContentComponent  {

        public myTemplate: any = '';
        
        constructor(private http: HttpClient) {
            http.get('file.html', 
                { responseType: 'text' })
                .toPromise()
                .then((html: any) 
                    => this.myTemplate = html);
        }        
    }

Serving Static HTML files through node server

1. You can use Connect and ServeStatic with Node.js for serving static simple html files via a custom node server

2. Install connect and serve-static with NPM

$ npm install connect serve-static

3. Create server.js file with this content:

var connect = require('connect'); var serveStatic = require('serve-static'); connect().use(serveStatic(__dirname)).listen(8080, function(){ console.log('Server running on 8080...'); });

4. Run with Node.js

$ node server.js

5. You can now go to http://localhost:8080/yourfile.html
